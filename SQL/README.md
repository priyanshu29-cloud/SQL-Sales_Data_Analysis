# 📊 Sales Data Analysis using Python

## 📖 Project Overview
This project focuses on analyzing sales data using Python. The data was cleaned, processed, and visualized to identify business trends and generate actionable insights.

## 🎯 Objectives
- Clean raw sales data
- Perform Exploratory Data Analysis (EDA)
- Identify sales trends
- Visualize important business metrics

## 🛠️ Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## 📊 Analysis Performed
- Missing Value Handling
- Duplicate Removal
- Data Cleaning
- Sales Trend Analysis
- Product Performance Analysis
- Customer Analysis

## 📈 Visualizations
- Sales Trend
- Monthly Revenue
- Top Products
- Category-wise Sales
