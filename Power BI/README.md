# 📈 Sales Dashboard using Power BI

## 📖 Project Overview
An interactive Power BI dashboard developed to monitor sales performance, revenue, and customer insights through dynamic visualizations.

## 🎯 Objectives
- Create interactive dashboards
- Track KPIs
- Monitor revenue trends
- Analyze customer behavior

## 🛠️ Tools Used
- Power BI
- DAX
- Power Query
- Excel

## 📊 Dashboard Features
- KPI Cards
- Sales Trend
- Revenue Analysis
- Category Analysis
- Filters & Slicers

## 📈 Business Insights
- Best Selling Products
- Highest Revenue Categories
- Monthly Growth
- Customer Segmentation
